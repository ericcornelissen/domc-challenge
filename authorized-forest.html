DOM Clobbering Challenge #5

Your goal: Get the site to load a new script named malicious.js.

One JavaScript pattern that is vulnerable to DOM Clobbering is when a global variable is used to store a configuration object. In the DOM, global variables are stored on the window object - which you already clobbered in challenge #1 - so you need to clobber using a similar approach.

However, in this scenario you need to clobber a specific configuration field, not just a property on the window object. In particular, you need to get the vulnerable script to load a script named malicious.js.

HINT: Which HTML element has a href attribute?
