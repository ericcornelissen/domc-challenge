DOM Clobbering Challenge #2

Your goal: Clobber document.domc

Secondly, you use DOM Clobbering to add and override properties of the document object (except url), which is the goal of this challenge. The document object can only be clobbered by certain elements. To clobber, you need to be able to insert a specific element into the DOM with the correct name attribute. The element must be one of <embed>, <form>, <iframe>, <img>, or <object>.
